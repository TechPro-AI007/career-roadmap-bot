An advanced, AI-powered interactive career counseling assistant built using **Botpress Studio** and deployed seamlessly via **GitHub Pages**. This web application helps aspiring professionals, students, and career-switchers discover personalized, step-by-step technical roadmaps, essential tools to master, and real-world job market insights.

---

## 🎯 Key Features

* **Custom Persona Alignment:** Driven by a tailored AI instruction block optimized for empathetic, structured, and realistic career guidance.
* **Dynamic Learning Roadmaps:** Generates structured learning phases (Beginner, Intermediate, Advanced) on-the-fly for any requested field (e.g., Cyber Security, Data Science, Full-Stack Development).
* **Tech Stack & Tool Guides:** Pinpoints the exact tools, programming languages, software, or industry-recognized certifications required.
* **Job Market Intelligence:** Highlights expected entry-level roles, target industries, and general hiring outlooks.
* **Responsive Dark UI:** Wrapped in a sleek, modern, glassmorphic dark-themed web wrapper to give a professional tech aesthetic.

## 🚀 How to Set Up & Deploy Your Own

If you want to duplicate or modify this deployment, follow these simple steps:

### 1. Build the Bot Logic
1. Create a workspace in **Botpress Studio**.
2. Create an autonomous node or standard text generation node and configure your custom AI persona prompt.
3. Turn off or remove any empty Knowledge Base lookup tools to allow the AI to rely directly on its master LLM instructions.
4. Hit **Publish** in the top right corner.

### 2. Grab Your Webchat Embed Code
1. Navigate to your Botpress Dashboard or **Deploy Settings** inside the workspace.
2. Under the **Embed Code** section, copy the two pre-configured `<script>` tags.

### 3. Deploy to GitHub Pages
1. Clone this repository or copy the code from `index.html`.
2. Replace the two `<script>` tags at the bottom of the `index.html` file with your unique Botpress embed script keys.
3. Push your changes back to your repository on GitHub.
4. Go to **Settings > Pages**, select the `main` branch, and click **Save**.
